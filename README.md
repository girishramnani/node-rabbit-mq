# node-rabbit-mq
A small tutorial to use rabbit-mq with nodejs


## Pre-requsites

- `Docker`
- `node`
- `npm`

## How to

- Install docker-compose from [here](git@github.com:girishramnani/node-rabbit-mq.git)
- `npm install`
- `docker-compose up -d`
- `node receive.js`
- open a new terminal and run `node send.js`
